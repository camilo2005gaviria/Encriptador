Encriptador de Soportes

Cifrado híbrido AES-256 + RSA · Compatible con Salud Total / FileZilla
Cifrados usados 0 / 100
Metodología híbrida: El archivo se cifra con AES-256-CBC (clave aleatoria), luego esa clave se cifra con RSA. Se generan dos archivos: .enc (contenido cifrado) y .enc.key (clave AES cifrada).
Clave Pública RSA Requerida para cifrar
Puedes pegar tu propia clave pública RSA (formato XML .NET), usar la clave de producción de Salud Total, o generar un par nuevo para pruebas.
